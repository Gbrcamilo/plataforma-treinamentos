# Plataforma de Treinamentos

Plataforma corporativa de treinamentos com áreas dedicadas para **Administrador**, **Gestor** e **Colaborador**.

## Sobre o projeto

Solução web para centralizar cursos, trilhas de aprendizagem, avaliações, certificados e indicadores de capacitação corporativa.

## Perfis

| Perfil | Responsabilidade |
|---|---|
| **Administrador** | Cadastro de cursos, trilhas, usuários, permissões e relatórios globais |
| **Gestor** | Acompanhamento da equipe, pendências, notas e alertas de vencimento |
| **Colaborador** | Acesso a cursos, trilhas, avaliações, certificados e progresso pessoal |

## Módulos planejados

- Catálogo de cursos
- Trilhas de aprendizagem
- Avaliações e quiz
- Certificados
- Indicadores e relatórios
- Notificações e alertas
- Gestão de usuários
- Biblioteca de materiais

## Fases de entrega

1. **Fase 1 — MVP**: autenticação, catálogo, trilhas simples e dashboard básico.
2. **Fase 2 — Gestão**: relatórios gerenciais, cobranças e validade de treinamentos.
3. **Fase 3 — Escala**: automações, importação em lote e certificados avançados.
4. **Fase 4 — Inteligência**: recomendação, ranking e metas por área.

## Stack sugerida

- **Frontend**: Vue 3 ou React com Vite
- **Backend**: Node.js com Express ou Fastify
- **Banco**: PostgreSQL ou MySQL
- **Auth**: JWT + RBAC (controle por papel)
- **Armazenamento**: Cloudinary ou S3 para materiais
